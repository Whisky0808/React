## What is higher order function and hoc?
- higher order function (HOF) is a general programming concept for a function that either accept one or more functions as an arguments or returns a function as its result. while (HOC) higher order component is a specific application of its pattern in React framwork.
## What is currying?
- a functional programming technique that transforms a function that takes multiple arguments into sequence of nested functions each taking a single argument. 
(a)=>(b)=>(c)=>a+b+c
## What/why custom hooks?
- custom hook is used for seperate the logic and reuse logic, clean up components
